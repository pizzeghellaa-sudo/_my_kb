Anthropic's official browsable gallery of Claude use cases at claude.com/resources/use-cases. Searchable and filterable by category. The standout pattern across many examples: Claude generating **interactive inline visualizations** as part of conversations — not just text, but live tools you manipulate mid-chat.

## Key Pattern: Inline Interactive Visualizations

Many use cases show Claude building interactive components inside the conversation:
- Scatter plots you place points on and watch curve-fitting respond
- Gift pyramids that rebuild when you drag a fundraising goal
- Budget scenario comparisons with toggleable dollar/percentage views
- Lit review argument maps with tension lines showing where camps disagree
- Correlation grids from uploaded CSVs with flags for patterns worth investigating

This is distinct from static charts — they're interactive reasoning tools generated in-context. [Source: Use Cases - Claude.url]

## Official Product Taxonomy (from page footer)

As of April 2026, Anthropic's Claude product lineup:
- **Claude** (consumer product)
- **Claude Code** — agentic coding tool
- **Claude Code for Enterprise**
- **Claude Cowork** — multi-agent/collaborative product
- **Claude for Chrome / Slack / Excel / PowerPoint** — integrations
- **Skills** — reusable skill files (see [[claude-code-skills]])
- **Opus / Sonnet / Haiku** — model tiers (see [[claude-model-selection]])

Plans: Pro, Max, Team, Enterprise. [Source: Use Cases - Claude.url]
